# Acira Website Redesign Plan
## Inspired by Bird.com & Framer Aesthetic

### Backup Location
`backups/original-design-20260106-234050/`

### Design Principles
- **Minimalism**: Clean layouts, generous whitespace
- **Bold Typography**: Large headings, clear hierarchy
- **Card-Based**: Structured sections with subtle shadows
- **Smooth Animations**: Fade-ups, parallax, micro-interactions
- **Dark Aesthetic**: Deep backgrounds with strategic pops of color

### Preserved Elements
✅ Color scheme (blues, gradients)
✅ Font families (var(--font-sans), var(--font-display))
✅ Cinematic laptop ad animation (27s timeline)
✅ Motion library (framer-motion)

### New Structure

#### 1. Navigation
- Floating translucent bar
- Minimal links
- Smooth backdrop blur
- Hide on scroll down, show on scroll up

#### 2. Hero Section
- Full viewport height
- Split layout: Large text left, Laptop mockup right
- Minimal badge/tag
- Large CTA button
- Scroll indicator

#### 3. Features Section
- 3-column card grid
- Hover lift animations
- Icons with gradient backgrounds
- Clean borders, subtle shadows

#### 4. "How It Works" Section
- Timeline/step-based layout
- Numbered circles
- Connecting lines
- Fade-in as you scroll

#### 5. Testimonials (Optional)
- Card carousel
- User avatars
- Star ratings
- Smooth transitions

#### 6. Pricing/CTA Section
- Centered card
- Clear value proposition
- Large CTA button
- Minimal pricing table (if needed)

#### 7. Footer
- Minimal links
- Social icons
- Back to top button

### Animation Enhancements
- Scroll-triggered fade-ups
- Parallax on hero
- Card hover effects with scale + shadow
- Smooth page transitions
- Micro-interactions on buttons
